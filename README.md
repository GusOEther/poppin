# Poppin

Poppin is a modern event discovery app built with React Native (Expo) and Firebase.

## Quick Start

1.  **Backend**: `./bin/dev-backend.sh` (Starts Firebase Emulators)
2.  **Frontend**: `./bin/dev-frontend.sh` (Starts Expo Web)
